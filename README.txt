Propagate It is allows pushing notificagtions across multiple clients over the web, which is deomonstrated by using websocket programming employing node.js and socket.io.

As per the requirements of the project, three buttons "door", "alarm" and "fire" are provided to the user logged in as server, which when clicked push the corresponding text to all the clients. Additionally, the user logged in server as the option to push custom text messages to all the clients by using the textbox provided.
